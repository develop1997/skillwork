## Get started

1. Install dependencies

    ```bash
    npm install
    ```

2. Run the app

    ```bash
     npm start
    ```

3. Build the app

    ```bash
     npm install --global eas-cli && eas init --id (Your app id)
     eas build --platform (android/ios)
    ```

    Note: You can get your app id from [expo.dev](https://expo.dev/)
